- 알람 종류

[cols=2*]
|===
|`OWNER_COMMENT`
|내가 올린 게시물에 댓글이 달렸을 경우

|`TAGGED_COMMENT`
|내가 태그된 게시물에 댓글이 달렸을 경우

|`COMMENT_REPLY`
|내 댓글에 답글이 달렸을 경우

|`FOLLOW`
|다른 사람이 날 팔로우했을 경우

|`OWNER_LIKE`
|내가 올린 게시물에 좋아요가 달릴 경우

|`TAGGED_LIKE`
|내가 태그된 게시물에 좋아요가 달릴 경우

|`TAG_APPROVAL`
|누군가 날 게시물에 태그했을 경우 (수락/거절 후 삭제되는 알람)

|===
